# הדרכת סימניות חכמה

## סקירה כללית
מערכת ההדרכה החדשה לסימניות מתבססת על הקשר ומוצגת רק כאשר התנאים הנכונים מתקיימים.

## מתי ההדרכה מוצגת
- המשתמש פותח את תפריט הסימניות
- קיימות סימניות ששמורות במערכת
- ההדרכה עדיין לא הוצגה למשתמש

## רכיבים
1. **bookmarksTutorial.ts** - מנוע ההדרכה העצמאי
2. **bookmarksMenu.ts** - אינטגרציה עם תפריט הסימניות
3. **תרגומים** - תמיכה באנגלית, יפנית וקוראנית

## איך זה עובד
1. כאשר המשתמש לוחץ על כפתור הסימניות
2. התפריט נפתח ונמלא בסימניות קיימות
3. אחרי 300ms המערכת בודקת אם להציג הדרכה
4. אם התנאים מתקיימים - מוצגת הדרכה קונטקסטואלית

## יתרונות
- לא מפריעה למשתמשים שלא צריכים אותה
- מוצגת רק כאשר יש תוכן רלוונטי
- תמיכה מלאה ברב-לשוניות
- חווית משתמש טבעית יותר

## טכנולוגיות
- TypeScript
- Browser Extension APIs
- CSS positioning
- browser.i18n for translations
